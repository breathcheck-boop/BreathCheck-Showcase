One product story: BreathCheck web app and PWA

BreathCheck helps users learn anxiety-management skills, use quick coping tools, and follow progress over time. The product runs in the browser and can also be installed as a PWA on desktop and mobile.

Install and Access

Install as PWA (Desktop or Mobile)

  1. Open the BreathCheck app in a modern browser.
  2. Use browser “Install app” or “Add to Home Screen”.
  3. Launch from desktop/home screen like a regular app.

After first load and cache warm-up, the app shell and core routes can still open offline.

Use in Browser

  • No installer package is required.
  • Works on desktop, tablet, and mobile browsers.
  • Each browser/device saves data locally first.

BreathCheck is local-first. Daily check-in ratings, tool summaries, and progress summaries can also sync online with a study participant ID when internet access returns.

Privacy and Safety

Local-first with participant-linked sync

Module written responses, support contacts, password-recovery details, and daily trigger text stay on the device. Daily check-in ratings, tool summaries, and progress summaries may also sync online with a study participant ID.

The research dataset does not include direct identifiers such as name, email, module written responses, support contacts, or daily trigger text.

Supportive, not emergency care

BreathCheck is for self-help and skills practice. It is not a crisis or emergency response service.
